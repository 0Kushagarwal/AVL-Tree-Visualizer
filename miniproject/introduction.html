Introduction

AVL Tree or Adelson-Velsky and Landsis Tree is a type of self balancing binary search tree. The AVL Tree is named after its two soviet inventores named Georgy Adelson-Velsky and Evgenii Landsis.

Key Points :-

  • The height difference between left nodes and right nodes of AVL Tree should not be greater than 1 or less than -1.
  • The range of the height difference must lie within the domain [-1,1] and be of type natural number.

Characterstics :-

  • AVL Tree is a type of Binary Search Tree so it follows all the properties of the Binary Search Tree.
  • Each subtree of the tree is balanced which states that the height difference between the left and right node must lie in a range of [-1, 1].
  • The tree balanced itself at the time of insertion or deletion of any node which makes insertion and deletion process time consuming.

Advantages :-

  1. AVL Tree can self balanced and gives us time complexity of O(log n) for searching, insertion and deletion of a node.
  2. It is a BST so the items can be traversed in a sorted way.
  3. AVL Tree in general have relatively less height and hence searching is faster.
  4. AVL Tree is relatively less complex to understand and implement as compared to few types of trees.

Disadvantages :-

  1. Compared to BST it is difficult to implement.
  2. Due to its stict balanced nature, the insertion or deletion of a node takes more number of rotations.
